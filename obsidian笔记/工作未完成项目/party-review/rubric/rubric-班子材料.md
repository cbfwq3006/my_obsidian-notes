# 2025年度基层党组织组织生活会 支部委员会对照检查材料 评分标准（Rubric）

## 评分总则
- 满分5分，最小刻度0.5分
- 7个维度加权计算总分
- 存在"硬伤"项直接扣分

---

## 维度一：结构完整性（权重15%）

评分标准：材料是否包含规定的全部结构要素。

| 必备要素 | 说明 |
|---------|------|
| 上年度组织生活会整改落实情况 | 须逐项报告整改进展 |
| 中央八项规定精神学习教育开展情况 | 须有具体做法 |
| 巡视巡察和审计整改情况 | 须有具体整改事项和结果 |
| 本年度问题查摆（6个方面） | 须覆盖全部6个方面 |
| 原因剖析 | 须从思想根源深挖 |
| 整改措施和努力方向 | 须与问题一一对应 |

6个方面查摆要求覆盖：
1. 加强理论武装
2. 执行上级组织决定
3. 严格组织生活
4. 加强党员管理监督
5. 做好群众工作
6. 抓好自身建设

| 分值 | 标准 |
|-----|------|
| 5 | 全部要素齐全，6个方面无遗漏，各部分篇幅均衡 |
| 4 | 全部要素齐全，6个方面无遗漏，个别部分篇幅偏少 |
| 3 | 缺少1个要素，或6个方面缺1个 |
| 2 | 缺少2个要素，或6个方面缺2个以上 |
| 1 | 结构严重残缺，缺少3个以上要素 |

---

## 维度二：问题查摆质量（权重30%）

这是最核心的维度，从以下4个子项综合评分。

### 子项A：见人见事（占本维度35%）
问题描述是否具体到人、具体到事、具体到后果，而非使用"部分""个别""有时"等模糊表述。

| 分值 | 标准 |
|-----|------|
| 5 | 每个方面至少1个具体事例（谁/什么事/什么时候/什么后果），无模糊主语 |
| 4 | 多数方面有具体事例，少量使用"部分""个别" |
| 3 | 有1-2个具体事例，但多数问题仍用模糊表述 |
| 2 | 无具体事例，全部使用"部分""个别"等模糊表述 |
| 1 | 问题表述空洞，无实质内容 |

### 子项B：问题深度（占本维度30%）
问题是否触及实质，是否有锐度，是否敢于揭短亮丑。

| 分值 | 标准 |
|-----|------|
| 5 | 问题直指要害，有自我揭短，能看出真实痛点 |
| 4 | 多数问题有一定深度，个别偏表面 |
| 3 | 问题中规中矩，不痛不痒 |
| 2 | 问题浮于表面，避重就轻 |
| 1 | 问题敷衍，明显回避实质 |

### 子项C：篇幅占比（占本维度15%）
问题查摆+原因剖析合计篇幅是否不少于全文的1/2。

| 分值 | 标准 |
|-----|------|
| 5 | 占比≥55% |
| 4 | 占比50%-54% |
| 3 | 占比40%-49% |
| 2 | 占比30%-39% |
| 1 | 占比<30% |

### 子项D：各方面均衡性（占本维度15%）
6个方面的查摆是否均衡，有无明显薄弱项。

| 分值 | 标准 |
|-----|------|
| 5 | 各方面均有2条以上问题，深度均衡 |
| 4 | 各方面均有问题，个别方面偏少 |
| 3 | 1-2个方面明显薄弱（仅1-2句话） |
| 2 | 3个以上方面明显薄弱 |
| 1 | 多数方面敷衍了事 |

### 子项E：禁止"先扬后抑"句式（占本维度5%）

明确禁止以下句式：
- "虽然……但是……"
- "虽能……但……"
- "尽管……却……"
- "虽……但……"
- 任何先肯定自己再引出问题的转折结构

| 分值 | 标准 |
|-----|------|
| 5 | 全文无先扬后抑句式，所有问题直接陈述 |
| 4 | 出现1处先扬后抑 |
| 3 | 出现2-3处 |
| 2 | 出现4-5处 |
| 1 | 大量使用，几乎每个问题都先肯定自己 |

---

## 维度三：原因剖析质量（权重15%）

| 分值 | 标准 |
|-----|------|
| 5 | 从思想根源深挖，紧密结合支部职能特点，有个性化分析，不套模板 |
| 4 | 能从思想根源分析，有一定个性化内容，但部分表述偏套路 |
| 3 | 基本从思想层面分析，但套路化明显，可套用到任何支部 |
| 2 | 剖析浮于表面，未触及思想根源 |
| 1 | 无实质性剖析，或直接复制模板 |

---

## 维度四：整改措施质量（权重15%）

| 分值 | 标准 |
|-----|------|
| 5 | 与问题一一对应，措施具体可操作，有责任人、时限、量化指标 |
| 4 | 基本与问题对应，多数措施具体，个别偏空 |
| 3 | 部分措施与问题对应，但有遗漏，部分措施是口号而非措施 |
| 2 | 措施与问题脱节，多数是空话套话 |
| 1 | 无实质性整改措施 |

---

## 维度五：上年度整改报告质量（权重10%）

| 分值 | 标准 |
|-----|------|
| 5 | 逐项报告整改进展，已完成项有成效说明，未完成项有原因分析和后续计划 |
| 4 | 逐项报告，但成效说明或原因分析不够具体 |
| 3 | 有报告但较笼统，未逐项说明 |
| 2 | 一笔带过，无实质内容 |
| 1 | 缺失 |

---

## 维度六：八项规定及巡视巡察整改（权重10%）

| 分值 | 标准 |
|-----|------|
| 5 | 内容详实，有具体做法和成效，体现支部特点 |
| 4 | 内容较详实，有具体做法 |
| 3 | 有内容但偏笼统 |
| 2 | 一笔带过 |
| 1 | 缺失 |

---

## 维度七：文字规范性（权重5%）

| 扣分项 | 扣分 |
|-------|------|
| 文号错误 | -0.5 |
| 错别字（每处） | -0.1，最多扣0.5 |
| 语句不通顺（每处） | -0.1，最多扣0.5 |
| 格式混乱 | -0.5 |

本维度基础分5分，扣完为止，最低1分。

---

## 总分计算公式

总分 = 维度一×15% + 维度二×30% + 维度三×15% + 维度四×15% + 维度五×10% + 维度六×10% + 维度七×5%

---

## 硬伤项（一票否决/强制扣分）

以下问题无论其他维度得分如何，总分上限受限：

| 硬伤 | 处理 |
|-----|------|
| 6个方面缺少2个以上 | 总分上限3分 |
| 全文无任何具体事例 | 总分上限3分 |
| 整改措施与问题完全脱节 | 总分上限3分 |
| 明显抄袭其他支部材料（大段雷同） | 总分上限2分 |
