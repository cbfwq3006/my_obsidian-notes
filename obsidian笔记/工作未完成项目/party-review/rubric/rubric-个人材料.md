# 2025年度组织生活会 个人发言提纲 评分标准（Rubric）

## 评分总则
- 满分5分，最小刻度0.5分
- 8个维度加权计算总分
- 存在"硬伤"项直接扣分

---

## 维度一：结构完整性（权重10%）

| 必备要素 | 说明 |
|---------|------|
| 上年度整改落实情况 | 逐项报告 |
| 八项规定精神学习教育 | 个人层面的落实 |
| 巡视巡察和审计整改 | 个人职责范围内的整改 |
| 六个方面问题查摆 | 全部覆盖 |
| 需要说明和报告的事项 | 配偶子女、房产、约谈函询等 |
| 原因剖析 | 思想根源 |
| 整改措施 | 与问题对应 |

6个方面查摆要求覆盖：
1. 学习贯彻党的创新理论
2. 加强党性锻炼
3. 联系服务群众
4. 发挥先锋模范作用
5. 改作风树新风
6. 宗教信仰及参加迷信活动

| 分值 | 标准 |
|-----|------|
| 5 | 全部要素齐全，6个方面无遗漏 |
| 4 | 全部要素齐全，个别部分偏简 |
| 3 | 缺少1个要素或1个方面 |
| 2 | 缺少2个以上要素 |
| 1 | 结构严重残缺 |

---

## 维度二：问题查摆质量（权重30%）

### 子项A：第一人称视角与个人聚焦（占本维度25%）

| 分值 | 标准 |
|-----|------|
| 5 | 全文以"我"为主语，所有问题聚焦个人行为，无"我们""支部"等集体主语 |
| 4 | 基本以"我"为主语，偶有集体表述 |
| 3 | 多处使用集体主语，个人与集体混杂 |
| 2 | 大量使用集体主语，个人视角模糊 |
| 1 | 基本是班子材料换了抬头 |

### 子项B：禁止"先扬后抑"句式（占本维度25%）

明确禁止以下句式：
- "虽然……但是……"
- "虽能……但……"
- "尽管……却……"
- "虽已……但……"
- 任何先肯定自己再引出问题的转折结构

| 分值 | 标准 |
|-----|------|
| 5 | 全文无先扬后抑句式，所有问题直接陈述 |
| 4 | 出现1处先扬后抑 |
| 3 | 出现2-3处 |
| 2 | 出现4-5处 |
| 1 | 大量使用，几乎每个问题都先肯定自己 |

### 子项C：见人见事与具体程度（占本维度30%）

| 分值 | 标准 |
|-----|------|
| 5 | 每个方面有具体事例（什么事/什么时候/什么后果），问题描述精准 |
| 4 | 多数方面有具体事例 |
| 3 | 有1-2个具体事例，其余偏笼统 |
| 2 | 无具体事例，全部概括性表述 |
| 1 | 问题空洞无物 |

### 子项D：问题深度与自我剖析锐度（占本维度20%）

| 分值 | 标准 |
|-----|------|
| 5 | 敢于自我揭短，问题触及个人思想深处，有真实的自我反省 |
| 4 | 多数问题有一定深度 |
| 3 | 中规中矩，不痛不痒 |
| 2 | 避重就轻，自我保护明显 |
| 1 | 敷衍了事 |

---

## 维度三：原因剖析质量（权重15%）

| 分值 | 标准 |
|-----|------|
| 5 | 从个人思想根源深挖，结合岗位特点，有真实的自我反省，不套模板 |
| 4 | 能从思想层面分析，有一定个性化 |
| 3 | 基本从思想层面分析，但套路化 |
| 2 | 浮于表面 |
| 1 | 无实质剖析 |

---

## 维度四：整改措施质量（权重15%）

| 分值 | 标准 |
|-----|------|
| 5 | 与6个方面问题一一对应（宗教信仰无问题可不对应），措施具体可操作，有时限 |
| 4 | 基本对应，多数措施具体 |
| 3 | 部分对应，有遗漏，部分措施空泛 |
| 2 | 措施与问题脱节 |
| 1 | 无实质措施 |

---

## 维度五：篇幅结构合理性（权重10%）

| 分值 | 标准 |
|-----|------|
| 5 | 问题查摆+原因剖析占全文≥50%，八项规定和巡视巡察部分简洁不喧宾夺主 |
| 4 | 占比45%-49%，结构基本合理 |
| 3 | 占比35%-44%，或八项规定/巡视巡察部分过长 |
| 2 | 占比25%-34%，成绩汇报喧宾夺主 |
| 1 | 占比<25% |

---

## 维度六：需要说明和报告的事项（权重5%）

| 分值 | 标准 |
|-----|------|
| 5 | 完整报告配偶子女情况、房产、经商办企业、约谈函询、党纪处分等 |
| 4 | 基本完整，个别项目缺失 |
| 3 | 有报告但不完整 |
| 2 | 过于简略 |
| 1 | 缺失 |

---

## 维度七：上年度整改及八项规定/巡视巡察（权重10%）

| 分值 | 标准 |
|-----|------|
| 5 | 上年度整改逐项报告，八项规定和巡视巡察部分简洁有重点，聚焦个人不足而非成绩汇报 |
| 4 | 内容较完整，但成绩汇报偏多 |
| 3 | 有内容，成绩汇报与问题检视各半 |
| 2 | 基本是成绩汇报 |
| 1 | 缺失或完全是成绩汇报 |

---

## 维度八：文字规范性（权重5%）

| 扣分项 | 扣分 |
|-------|------|
| 文号错误 | -0.5 |
| 错别字（每处） | -0.1，最多扣0.5 |
| 语句不通顺（每处） | -0.1，最多扣0.5 |
| 与班子材料大段雷同 | -1.0 |

基础分5分，扣完为止，最低1分。

---

## 总分计算公式

总分 = 维度一×10% + 维度二×30% + 维度三×15% + 维度四×15% + 维度五×10% + 维度六×5% + 维度七×10% + 维度八×5%

---

## 硬伤项（强制扣分）

| 硬伤 | 处理 |
|-----|------|
| 6个方面缺少2个以上 | 总分上限3分 |
| 全文无"我"，以集体视角写作 | 总分上限2.5分 |
| 先扬后抑句式≥5处 | 总分上限3.5分 |
| 整改措施与问题完全脱节 | 总分上限3分 |
| 与班子材料大段复制粘贴 | 总分上限2.5分 |
| 需要说明和报告的事项完全缺失 | 总分扣0.5分 |
