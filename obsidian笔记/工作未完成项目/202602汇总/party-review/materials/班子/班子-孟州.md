2025年度基层党组织组织生活会

孟州分公司支部委员会对照检查材料

焦作联通孟州分公司党支部

按照《关于召开基层党组织2025年度组织生活会和开展民主评议党员的通知》(焦联通党委〔2026〕10号)要求，党支部班子结合工作实际，认真对照查摆，深挖问题症结，深刻进行党性分析，有针对性地提出改进措施。现对照检查发言如下：

一、上年度组织生活会整改落实情况

1.贯彻落实上级工作安排不够到位。比如：落实基层责任单元党建负责人制度推动执行不到位，非合同制用工党员管理仍需加强。支委会记录不够规范，个别普通党员未按要求记录三会一课会议记录。

严肃党内政治生活，规范党务基础工作，提高“三会一课”、主题党日和组织生活会质量，发挥党支部引领、凝聚、教育党员的战斗堡垒作用，落实好基层责任单元党建负责人制度，建立分包党员台账，通过基层责任单元建立联系帮扶机制，网格结对共建等方式，将党员下沉到基层责任单元，切实让党员走在前列、干在实处，团结带领基层责任单元员工群众奋勇争先、攻坚克难。加强对公司非合同制党员管理，2025年对非合同制党员进行排查，规范党员组织关系和档案规范管理。

2. 党风廉政建设需进一步加强。如二度党风廉政建设有扣分，请落实整改。

认真落实党风廉政建设主体责任，严格履行一岗双责，坚持把纪律和规矩挺在前面，常态化开展警示案例教育、廉洁风险防控、小CEO约谈、纪律教育宣传月等工作，通过宣传展板、廉洁视频等形式丰富学习形式；二是深化以案促改，推动基层网格“十条铁律”、十五项禁止的落地执行，不断强化干部职工廉洁从业意识，锤炼队伍敢打硬仗的作风，筑牢拒腐防变思想防线。

3. 巡察整改存在制度落实不到位。如评优评先未在支委会上讨论，被扣分。

认真落实“三重一大”决策制度，进一步规范党支部议事决策程序，在考核办法制定、干部考核任命、重大资金支出等关键事项有效发挥党支部把方向、管大局、保落实的作用，不断提升支部政治能力。

4. 支部书记述职工作方面要进一步提升质量，加强理论学习的系统性和精准度，深化基层党建“三基”建设，强化问题剖析深度与数据支撑，以高质量党建引领孟州公司实现高质量发展。

把党的最新理论与实际工作结合起来，着力破解高质量发展中的难题，全面提升工作质效。二是提高“政治三力”不断提高把握新发展阶段、贯彻新发展理念、构建新发展格局的政治能力、战略眼光、专业水平，敢于担当、善于作为，把集团1+9+3战略决策部署贯彻落实好。三是充分发挥头雁效应，带头深入一线开展调查研究，解决痛点、堵点、难点，切实为一线减负。四是加强履职尽责，夯实主责主业。扛牢国企担当，围绕公司年度收入目标及重点工作，强化责任意识，提升攻坚克难的勇气和担当作为的豪气，把“钉钉子”精神落到实处，把直面挑战的毅力体现在实际行动中，坚定不移推动公司高质量发展提能升级。

二、深入学习贯彻中央八项规定精神学习教育开展情况

一、主要做法

（一）深化学习研讨，夯实思想根基

1.支部召开专题会议，精心制定《中央八项规定精神学习教育实施方案》，明确学习目标、任务分工及时间节点，确保学习教育有序推进。

2.聚焦中央八项规定原文、典型案例及集团内部制度文件，组织8场“中央八项规定精神”专题学习会。采用“线上+线下”相结合的方式，开展党章党规、典型案例解读等专题培训，实现全员参训。

3.深入学习习近平总书记考察河南重要讲话精神，特别是关于贯彻中央八项规定精神的要求。以读书班、“三会一课”、主题党日、总经理讲党课等形式，组织党员干部集中学习研讨，重点学习中央禁止违规吃喝规定及省委“十禁止”要求。

（二）全面查摆问题，精准靶向纠治

1.对标对表查：对照习近平总书记关于作风建设的重要论述、中央八项规定及其实施细则精神，以及《中国共产党章程》《中国共产党纪律处分条例》等规定，结合《中国联通整治形式主义为基层减负七条举措》《中国联通河南省分公司认真贯彻落实中央八项规定精神坚决纠正“四风”问题实施细则》等制度文件，全面查摆问题。

2.多措并举查：充分运用纪检委员监察、兼职纪检员巡视巡察、信访反映等多种渠道，聚焦学习教育主题，筛查分析突出问题，确保问题无遗漏。

3.结合案例查：开展以案说德、以案说纪、以案说法、以案说责活动，结合本单位或本领域、系统内典型案例，加强警示教育。通过剖析反面典型，深挖问题根源，引导干部职工对照查找自身差距。

4.以上率下查：公司支委、管理部门、领导干部以身作则，带头从严查摆问题，形成一级抓一级、一级带一级的良好局面，为广大党员树立榜样。

（三）强化整改整治，巩固作风建设成果

1.制定整改清单：针对作风问题在不同层面的表现，深入梳理分析，建立问题清单，逐项制定整改措施，实行挂账督办、对账销号，确保整改到位。

2.严肃党内政治生活：规范党务基础工作，提升“三会一课”、主题党日和组织生活会质量，发挥党支部战斗堡垒作用。开展谈心谈话活动，关心干部职工思想动态、家庭生活和社会交往，及时发现和解决问题。建立科学有效的考核激励和奖惩约束机制，营造真抓实干、干事创业的良好氛围。

3.建立长效机制：坚持“解决一个问题、堵塞一个漏洞、完善相关制度”的原则，在集中整治的同时，深挖问题根源，完善贯彻中央八项规定及其实施细则精神的制度规定，建立作风建设常态化长效化机制，巩固整治成效。

四、本年度党支部查摆问题情况

党支部重点从加强理论武装、执行上级组织决定、严格组织生活加强党管理监督、做好群众工作、抓好自身建设等6个方面查摆突出问题、找准思想根源。

(1)加强理论武装方面。

1. 理论学习不够系统深入，学习碎片化、表面化，对党的创新理论理解不透彻。

2. 学习形式单一，多以念文件、读材料为主，研讨交流不深入，学习效果不明显。

3. 学用结合不够紧密，理论指导实践、推动工作的作用发挥不充分。

4. 学习制度执行不严，计划性、持续性不足，存在重业务、轻学习现象。

(2)执行上级组织决定方面。

1. 政治执行力不够强，对上级决策部署理解不深、落实不够坚决。

2. 存在重部署、轻落实现象，跟踪问效不到位，推进力度层层递减。

3. 攻坚克难劲头不足，面对难点问题存在等靠观望思想。

4. 请示报告制度执行不够严格，重大事项汇报不及时。

(3)严格组织生活方面。

1. 组织生活制度落实不够规范，“三会一课”质量不高，存在简化程序现象。

2. 批评与自我批评辣味不足，避重就轻，一团和气，未达到红脸出汗效果。

3. 谈心谈话不深入，谈工作多、谈思想少，交流不够坦诚。

4. 组织生活形式单一，吸引力不强，党员参与积极性有待提高。

5. 组织生活记录不规范，痕迹管理不够严谨。

(4)加强党员管理监督方面。

1. 党员日常教育管理不够严格，抓早抓小、防微杜渐做得不够。

2. 对党员监督提醒不够经常，失之于宽、失之于软。

3. 流动党员、年老体弱党员管理服务不到位，存在薄弱环节。

4. 民主评议党员不够严格，结果运用不充分。

(5)做好群众工作方面。

1. 联系群众不够紧密，深入基层、深入群众不够经常。

2. 服务群众意识不够强，解决群众急难愁盼问题不够及时。

3. 倾听群众呼声、收集意见建议渠道不够畅通。

4. 为民办实事力度不够，成效不够明显，群众获得感不强。

(6)抓好自身建设方面。

1. 支部班子整体功能发挥不够充分，凝聚力、战斗力有待提升。

2. 制度建设不够完善，执行不够到位，内部管理不够规范。

3. 班子成员带头示范作用不够明显，领学领干能力不足。

4. 工作创新意识不强，亮点不多，特色不突出。

五、原因剖析

(一)在加强理论武装方面

1. 思想认识不到位，政治站位不够高，把理论学习当作软任务。

2. 主动学习意识不强，缺乏深学细悟的自觉性和紧迫感。

3. 对学习成果转化思考不多、研究不深，存在学做脱节问题。

(二) 在执行上级组织决定方面

1. 大局意识不够强，存在本位主义思想，执行缺乏刚性。

2. 责任担当不足，主动作为、狠抓落实的劲头不够。

3. 督办机制不健全，跟踪问效、闭环管理不到位。

(三)在严格执行组织生活方面

1. 对组织生活的政治性、严肃性认识不足，重视程度不够。

2. 党性锻炼不够严格，缺乏真刀真枪开展批评的勇气。

3. 创新意识不足，方式方法陈旧，缺乏针对性和实效性。

(四)在加强原因剖析党员管理监督方面

1. 严管厚爱的意识树得不牢，重使用、轻监管。

2. 日常监督机制不健全，缺乏常态化、制度化约束。

3. 分类管理意识不强，措施不够精准。

(五) 在做好群众工作方面

1. 宗旨意识有所淡化，群众立场站得不够稳。

2. 工作方法较为传统，主动服务、靠前服务不够。

3. 以人民为中心的发展思想树得不够牢固。

(六)在抓好自身建设方面

1. 自身建设重视不够，标准不高、要求不严。

2. 责任压得不够实，整体合力没有充分激发。

3. 进取精神不足，缺乏创先争优意识。

六、整改措施和努力方向

(一)在加强理论武装方面

1. 制定年度学习计划，坚持集中学习与个人自学相结合，做到每周有学习、每月有研讨。

2. 丰富学习形式，开展专题党课、现场教学、交流研讨，提升学习吸引力。

3. 坚持学用结合，把理论学习与中心工作紧密结合，推动学习成果落地见效。

4. 严格落实学习制度，强化督促检查，确保学习不走过场、取得实效。

(二)在执行上级组织决定方面

1. 提高政治站位，对上级决定第一时间传达、不折不扣执行。

2. 建立任务清单、责任清单，明确时限、逐项落实、逐项销号。

3. 强化担当精神，主动破解难题，杜绝推诿扯皮、敷衍应付。

4. 严格执行请示报告制度，重要工作、重大问题及时向组织汇报。

(三) 在严格执行组织生活方面

1. 严格规范落实“三会一课”、组织生活会、主题党日等制度。

2. 提高批评与自我批评质量，敢于揭短亮丑，增强党内政治生活“辣味”。

3. 常态化开展谈心谈话，做到必谈、深谈、实谈，及时掌握思想动态。

4. 创新组织生活形式，丰富活动内容，提升党员参与度。

5. 规范会议记录，做到要素齐全、内容详实、留存完整。

(四)在加强党员管理监督方面

1. 强化日常教育管理，严格党员教育、管理、监督全链条。

2. 坚持抓早抓小，对苗头性问题及时提醒、约谈、纠正。

3. 加强分类管理，做好流动党员联系、年老党员送学服务。

4. 规范民主评议党员工作，强化结果运用，树立鲜明导向。

(五)在做好群众工作方面

1. 牢固树立宗旨意识，常态化开展联系群众、走访群众活动。

2. 畅通意见渠道，及时收集、回应、解决群众合理诉求。

3. 扎实开展为民办实事活动，解决群众实际困难。

4. 改进工作作风，提升服务质效，增强群众满意度。

(六)在抓好自身建设方面

1. 加强支部班子建设，提升政治功能和组织功能。

2. 健全完善各项制度，强化制度执行，做到用制度管人管事。

3. 班子成员带头示范，以上率下，发挥好头雁效应。

4. 强化创新意识，打造工作亮点，推动支部建设提质增效。
