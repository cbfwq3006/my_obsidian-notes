2025年度基层党组织组织生活会

创新业务支撑中心支部委员会对照检查材料

焦作联通创新党支部

按照《关于召开基层党组织2025年度组织生活会和开展民主评议党员的通知》（焦联通党委〔2026〕10号）要求，党支部班子结合工作实际，紧扣会议主题，在广泛征求意见、开展谈心谈话的基础上，深入查摆问题，深刻剖析根源，明确整改方向。现对照检查如下：

一、上年度组织生活会整改落实情况

1、政治纪律和规矩执行方面整改情况

针对“对纪律内涵理解不深，学习满足于一般性了解”的问题

整改措施：开展“纪律规矩深学细悟”专项行动。创新中心党支部组织全体党员共同学习《中国共产党纪律处分条例》，会后要求每位党员结合岗位实际撰写心得体会。

落实情况：已完成。党员对政治纪律的具体条款知晓率达到100%，实现了从“一般性了解”到“深层次把握”的转变。

针对“维护党的团结统一方面沟通协作存在障碍，存在信息壁垒”的问题：

整改措施：建立跨行业的商机互通与业务协同机制，每位党员分包有县区，也分包行业，通过分包负责人每周五梳理汇总分包县区商机要点并精准分配至对应行业负责人深度跟进；全面开展“谈心谈话全覆盖”活动，确保支委之间、党员之间每季度至少进行一次深度沟通，以消除信息壁垒、增进内部互信。

落实情况：已完成。工作中推诿扯皮现象基本消失。

针对“长效学习机制落实不到位，集中学习走过场”的问题

整改措施：优化个人考核评价体系。创新中心建立个人积分卡，个人学习、培训赋能、陪访、生活馆接待、编写案例和方案撰写等，每完成一个积1分，倒逼党员由“被动听”转为“主动讲”。

落实情况：已完成。个人积分考核应用12个月,从月初积分相差6分到后面逐步追齐，党员积极性显著提高，得到根本扭转。

针对“‘两个确立’‘两个维护’践行转化不足，缺乏大局意识”及“面对错误言论缺乏斗争精神”的问题

整改措施：为切实将政治要求转化为行动自觉，将大局意识贯穿工作始终，中心实施分小组责任制,总分为4组，为标品组、IDC组、项目组、标书组，并设立小组长，由组长对分管业务进行牵头负责，梳理业务内容，规范业务管理。

落实情况：已取得阶段性成效。项目组已实现从“前置营销”的单点思维向“全流程跟进”的系统思维转变；标书组标书组强化审核把关，政治意识与业务水平实现双提升。

2、党性修养方面整改情况

针对“党性教育活动流于形式，理论融入决策不足”的问题

整改措施：推行“沉浸式”党性教育+支部书记讲党课。开展“重温入党誓词”及政治生日活动。支部书记带领大家将理论结合业务场景深层次讲解。

落实情况：已完成。2025年12月完成支部书记讲党课。

针对“作风建设存在形式主义，服务群众缺乏耐心”的问题

整改措施：推行“首问负责制”，将本部创新支撑经理全部融入行业网格，与网格业绩强关联，建立同奖同罚机制；同时人员分包县区，一人一区县，一事贯穿到底原则。

落实情况：已完成。

针对“权力运用不够民主、不接地气”及“价值观践行不够持续，奉献精神不足”的问题

整改措施：完善民主决策与利益协调机制。严格执行民主集中制，重大事项决策前必须广泛征求群众意见。

落实情况：已完成。

3、担当创新方面整改情况

针对“面对复杂任务畏难情绪重，习惯旧模式”及“矛盾问题应对拖延回避”的问题

整改措施：开展“创新案例评选”，奖励大胆尝试新方法并取得实效的团队。

落实情况：已完成。

针对“政绩观存在偏差，重表面轻实效”及“风险防范敏锐性不足”的问题

整改措施：树立正确政绩观与强化风险预警。修订绩效考核办法，降低过程性指标权重，增加实效性、长远性指标权重。

落实情况：已取得阶段性成效。工作导向更加务实。

4、落实改革精神和严标准管党治党方面整改情况

针对“制度执行不严，‘三会一课’质量不高，批评与自我批评‘走过场’”的问题

整改措施：实施“组织生活规范化”提升行动。

落实情况：已完成。组织生活规范性显著增强，真正达到了红脸出汗、排毒治病的效果。

针对“纪律教育形式单一”及“监督执纪‘四种形态’运用不充分”的问题

整改措施：创新教育载体与从严执纪问责。开展“以案促改”警示教育，用身边事教育身边人。

落实情况：已完成。

针对“引导党员在网络空间主动发声、敢于斗争缺乏系统培训”的问题

整改措施：构建网络意识形态工作体系。引导党员理性发声、敢于亮剑。

落实情况：需持续深化。面对复杂多变的网络舆情，党员的斗争策略和技巧仍需在实践中进一步磨练和提升。

总结：

通过一年的集中整改，支部在政治建设、思想建设、组织建设、作风建设和纪律建设等方面取得了明显进步，班子凝聚力战斗力显著增强。下一步，支部将继续坚持“当下改”与“长久立”相结合，对已完成的整改任务适时开展“回头看”，防止反弹回潮；对需持续深化的问题，紧盯不放、久久为功，确保整改工作经得起时间和群众的检验。

二、深入学习贯彻中央八项规定精神学习教育开展情况

2025年以来，党支部始终将深入贯彻落实中央八项规定及其实施细则精神作为加强作风建设的“铁规矩”和“硬杠杠”，坚持严字当头、实字托底，通过“学、查、改、立”全链条发力，推动作风建设向纵深发展。

三、本年度党支部查摆问题情况

党支部重点从加强理论武装、执行上级决定、严格组织生活、加强党员管理监督、做好群众工作方面、抓好自身建设等6个方面，协助支部班子深挖细查，共查摆出突出问题10个。

1. 加强理论武装方面

一是学习转化存在“温差”。支部在引导本部创新支撑经理时，未能有效督促其将学习成果向下延伸，导致部分同志虽然熟悉产品知识、PPT材料和政策文件，但在实际工作中存在“照本宣科”现象，未能真正触及一线人员的痛点难点，学习归学习、业务归业务，理论与实际结合不够深入。

二是学思用贯通不够。本部95后群体较多，思维活跃、价值多元，但支部层面缺乏有针对性的思想引导，未能将理论武装与年轻人的所思所想紧密结合；谈心谈话只谈工作不谈思想，导致未能真正走进内心，致使思想政治工作的感召力和穿透力不足，学思用贯通在支部层面存在“断档”。

2. 执行上级组织决定方面

一是战略传导存在“层层递减”现象。支部承接市公司战略部署时，个别支委成员对战略意图研究不深，在向部门传达时，只强调考核指标不说背景，导致战略落地的“颗粒度”不够精细。例如，在拓展“智算”专项活动上，部分党员只知道“要做什么”，却不知道“为什么做”以及“怎么做标准”，导致执行层面效率低下。

3. 严格组织生活方面

一是“三会一课”质量不高。存在“重数量、轻质量”问题，能够做到按时上传，存在上传缺少材料导致不过关的情况，如台账或者其他材料等。

二是组织生活会“辣味”不足。支部开展批评与自我批评时，存在“老好人”思想，多以工作建议代替思想交锋。比如对创新支撑经理只懂产品不懂一线的现象，会上只提“要加强调研”，却未深究其一线真正需要什么；对年轻员工思想活跃但无人引导的问题，只讲“要多关心”，不敢点透缺乏主动学习能力。直指党性修养、纪律作风的尖锐问题少，导致红脸出汗成为走过场，未能真正触及灵魂、解决问题。

4. 加强党员管理监督方面（组织委员重点查摆）

一是党员日常教育管理“宽松软”。支部对党员的管理局限于工作圈，对党员八小时外的监督管理存在盲区。比如本部人员在办公室呆的时间要长于下一线时间，导致有时候脱离实际，制定的解决方案停留着表面，未深究直击客户痛点，导致项目存在流失风险。

二是运用“四种形态”不够主动。支部对党员出现的苗头性问题睁一只眼闭一只眼。比如对年轻员工思想活跃但无人引导的现象，明知欠缺主动学习，却从未及时提醒谈话，错过了抓早抓小的最佳时机。

# 5、做好群众工作方面

一是群众工作方法简单。 在做群众思想工作时，习惯讲大道理，缺乏同理心。

6.  抓好自身建设方面

一是班子团结协作有欠缺。支委之间沟通协作不够，存在“各自为战”现象。由于创新中心支撑人员15人，4个小组分包制，同时部分人员还要承担专项活动，导致会存在某个节点一个人同时收到2-3个工作安排，疲于应付，也侧面暴漏出缺乏统筹配合，导致党建工作合力不足。

二是作风不实、纪律松弛。存在“老好人”思想，对支部内存在的慵懒散漫现象不敢较真碰硬。支委成员带头示范作用发挥不够，要求党员做到的，自己有时未能率先垂范。

五、原因剖析

针对上述查摆出的问题，支部委员会从思想深处进行了深刻剖析，主要原因如下：

（一）政治站位不够高，理论武装“虚化”。

支委班子对政治理论学习的极端重要性认识不足，存在“重业务轻党建”的错误思想。片面认为只要业务工作干好了，就是最大的政治，导致理论学习沦为“软任务”。没有真正把学习贯彻习近平新时代中国特色社会主义思想作为提升组织力的根本途径，缺乏从政治高度审视和组织建设工作的自觉性，导致学用脱节，理论指导实践的能力不强。比如创新支撑经理，虽然产品理论、政策文件学得不少，但缺乏与一线人员的深度沟通，不了解客户和客户经理要什么？坐在办公室搞支撑、对着PPT讲产品，看似理论武装“学过了”，实则服务群众的宗旨意识“虚化了”。支部未能引导党员干部从政治高度认识“深入一线”的重要性，导致学用“两张皮”。

（二）责任担当不够强，斗争精神“弱化”。

面对全面从严治党的高标准严要求，支委班子存在畏难情绪和“守摊子”思想。尤其创新支撑经理存在靠前意识不足，前置支撑力度不够。

（三）制度建设不够严，执行力度“软化”。

对制度执行情况的监督检查不到位，对违反制度的行为追责问责不力，导致制度失去了应有的严肃性和权威性，形成了“破窗效应”。比如创新支撑经理绩效与项目利益联动弱，内生动力未能有效激发。

（四）创新意识不够足，工作方法“僵化”。

习惯于凭经验办事，沿袭老套路、老办法，缺乏主动创新的意识和能力，面对新问题束手无策，工作成效不明显。比如，算网数智毛利较低，仍按照传统方式拓展项目，融自研能力不足；创新支撑团队解决方案自主交付能力不足，也制约了一线商机获取效率与签约转化成效。

# （五）宗旨意识不够牢，群众观念“淡化”。

# 随着工作环境的变化，支委成员深入群众的时间少了，听汇报多了。潜意识里认为自己是管理者而非服务者，对群众的诉求回应不快。在谋划工作时，更多考虑上级满不满意，较少考虑群众答不答应，导致服务群众流于形式。

（六）自身建设不够硬，表率作用“退化”。

支委班子成员自我要求有所放松，艰苦奋斗的精神有所懈怠。在日常工作和生活中，未能时刻以优秀共产党员的标准严格要求自己。比如，创新支撑经理自身知识理论不扎实，眼界不够开阔，无法与新质生产力、新型工业化、低空经济等最前沿技术理论紧密结合。

六、整改措施和努力方向

针对查摆出的问题和剖析的原因，支部委员会将坚持问题导向，制定如下整改措施，确保件件有着落、事事见成效。

（一）强化理论武装，在“学懂弄通做实”上下功夫。

具体措施：

优化“学分制”考核机制。将党员理论学习情况量化积分，每季度公示，对排名靠后的党员由组织委员进行约谈；强化每天本部内部培训、每周全区创新支撑团队培训、每月“送陪到基层”培训、每季度“走出去”到产互、云洹、优秀案例实地参观学习。

责任人：支部书记、组织委员；完成时限：即知即改，长期坚持。

（二）严守政治纪律，在“令行禁止”上见真章。

具体措施：

建立“即时响应”机制。对县区、3中心递交的商机，确保1天内对接需求、3天内整理出具解决方案、1周内完成现场调研；对市公司决策部署，24小时内制定落实方案，明确时间表、路线图，实行挂图作战。严格遵守国资委“十不准”要求，加强业务合规，从项目分包、标前评审、实施交付、项目计收等环节加强管控。

责任人：全体支委；完成时限：2025年6月底前见效。

（三）规范组织生活，在“提质增效”上出实招。

具体措施：

提高组织生活会质量。会前广泛征求意见，会上直奔主题、揭短亮丑，会后建立整改台账，实行销号管理。强化对创新产品交付的标准化管理与产品支撑，做好行业洞察、项目台账建立、紧密联系县区全流程工作。

责任人：组织委员；完成时限：长期坚持。

（四）从严管理党员，在“抓早抓小”上求突破。

具体措施：

用好“第一种形态”。建立常态化谈心谈话制度，发现苗头性问题及时提醒，让“红脸出汗”成为常态。

责任人：组织委员、纪检委员；完成时限：2025年底前完善机制。

# （五）践行群众路线，在“排忧解难”上显担当。

# 具体措施：

# 定期深入了解，切实解决实际困难。

# 责任人：宣传委员、组织委员；完成时限：长期坚持。

（六）加强自身建设，在“示范引领”上树标杆。

具体措施：

建立“支委履职清单”。明确每位支委的职责分工，实行季度述职评议，接受党员群众监督。支委根据专项行动分包到人，成立企业、人工智能、低空经济专班，专项支撑提升能力建设。

开展“作风建设年”活动。带头遵守各项规章制度，设立“党员先锋岗”，以实际行动感召群众。规范新兴ICT、招投标、标前评审等各项流程，确保项目正常推进。

责任人：支部书记；完成时限：2025年全年推进。

党支部将以此次组织生活会为新起点，以更坚定的信念、更务实的作风、更有力的举措，抓好整改落实，切实把党支部建设成为坚强战斗堡垒，为推动单位高质量发展提供坚强组织保证。
