# Raw 原始资料目录

> 本目录存放所有原始资料，包括文章、论文、笔记、网页剪藏等。
> **重要**：此目录由用户管理，Claude 只读不写。

---

## 使用说明

### 添加资料

1. 使用 **Obsidian Web Clipper** 插件一键保存网页内容
2. 手动复制粘贴文章到 `.md` 文件
3. 上传 PDF、图片等文件

### 图片管理

- 所有图片统一存放在 `raw/素材/` 目录
- 使用 Obsidian 的 "Download attachments" 功能下载网页图片到本地
- 在文章中用 `![[素材/图片名.png]]` 引用

### 文件命名建议

- 使用有意义的文件名，如：`2026-04-19 Karpathy Wiki 方法论.md`
- 避免使用特殊字符和空格
- 可以使用中文命名
