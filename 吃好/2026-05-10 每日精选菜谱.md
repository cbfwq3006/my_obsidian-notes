## 1. 香菇滑鸡

- 配图：
- 来源链接：https://www.xiachufang.com/recipe/104116894/
- 推荐理由：鸡腿肉+香菇的组合非常家常，做法集中在“腌→炒→焖收汁”，汤汁拌饭很友好；整体可用少油完成，适合晚餐与便当。
- 适合场景：工作日晚餐主菜 / 盖饭浇头 / 便当备餐
- 预计耗时：25–35 分钟（含泡香菇、腌制）
- 主要食材：鸡腿（去骨更省事）、干香菇、葱姜蒜、生抽、料酒、淀粉
- 简明步骤：
  1) 干香菇温水或热水泡软（泡菇水留用）；鸡腿去骨切丁，加生抽/料酒/淀粉/少许盐腌 20–30 分钟。
  2) 锅中少油，鸡丁下锅快速炒至变色盛出；原锅爆香葱姜蒜，下香菇炒匀。
  3) 回锅鸡丁，少量加入泡菇水/清水，翻匀焖 2–3 分钟收汁即可。
- 营养亮点：优质蛋白（鸡腿）+ 菌菇类（香菇）增加膳食纤维与风味；配蔬菜可做更均衡的一餐。
- 注意事项：香菇泡得不够软会偏硬；鸡丁先炒后回锅更嫩；生抽本身含盐，最后再视情况补盐。
- 自动筛选依据：下厨房独立菜谱页；口碑长期稳定的家常组合；步骤集中、食材常见、可少油；与 吃好/ 和 memory 去重索引无重复。

## 2. 番茄烧豆腐

- 配图：
- 来源链接：https://www.xiachufang.com/recipe/105835355/
- 推荐理由：番茄酸甜开胃，搭配豆腐更“清爽但下饭”；整体调味简单、步骤少，适合晚餐或备餐。
- 适合场景：清淡晚餐主/副菜 / 控油日 / 配米饭或面条
- 预计耗时：20–30 分钟
- 主要食材：北豆腐、番茄、蒜、葱（可选）、番茄酱（可选）、生抽/蚝油少量
- 简明步骤：
  1) 番茄划十字焯水去皮切块；豆腐切块沥干水分。
  2) 豆腐中小火少油煎至微金黄；蒜末下锅爆香，下番茄炒出汁。
  3) 加少量番茄酱/生抽/蚝油调味，下豆腐小火焖 2–3 分钟入味，出锅撒葱花。
- 营养亮点：植物蛋白（豆腐）+ 番茄的维生素与番茄红素；适合搭配一份绿叶菜补纤维。
- 注意事项：豆腐先沥水更不易碎；调味里有生抽/蚝油时注意控盐；不想煎也可改为轻焯后直接焖。
- 自动筛选依据：下厨房独立菜谱页；家常快手、可少油、食材易买；与 吃好/ 和 memory 去重索引无重复。

## 3. 木耳炒鸡蛋

- 配图：
- 来源链接：https://www.xiachufang.com/recipe/106480902/
- 推荐理由：一盘“蛋白质+膳食纤维”的家常快手菜；木耳泡发后下锅翻炒时间短，适合下班快速出菜。
- 适合场景：工作日晚餐快手菜 / 便当配菜 / 配粥配饭
- 预计耗时：15–25 分钟（含泡发）
- 主要食材：干木耳、鸡蛋、生抽、米醋（可选）、蚝油（可选）、淀粉（可选）
- 简明步骤：
  1) 木耳热水泡发洗净沥干；鸡蛋打散加少许盐（可滴几滴白醋/柠檬汁让蛋更嫩）。
  2) 锅中少油炒蛋至八成熟盛出；原锅下木耳快炒 1 分钟。
  3) 回锅鸡蛋，按口味加少量生抽/蚝油，快速翻匀出锅（可用少量水淀粉让酱汁挂味）。
- 营养亮点：鸡蛋提供优质蛋白；木耳提供膳食纤维，口感脆爽、饱腹感好。
- 注意事项：木耳泡发后要充分冲洗；炒制时间不宜过久避免出水；酱油/蚝油含盐，注意总盐量。
- 自动筛选依据：下厨房独立菜谱页；步骤短、食材常见、少油可做；与 吃好/ 和 memory 去重索引无重复。

## 4. 手撕包菜

- 配图：
- 来源链接：https://www.xiachufang.com/recipe/1016863/
- 推荐理由：经典家常快手素菜，包菜便宜易买；大火快炒几分钟就能出锅，适合搭配任何主菜。
- 适合场景：2 菜 1 汤里的蔬菜位 / 快手晚餐 / 备餐蔬菜
- 预计耗时：10–15 分钟
- 主要食材：包菜、蒜（可选）、花椒/干辣椒（可选）、生抽、少许糖
- 简明步骤：
  1) 包菜手撕成片，尽量沥干水分。
  2) 热锅少油，花椒/蒜/干辣椒爆香。
  3) 下包菜大火快炒至变软，淋少量生抽，少许糖提味，翻匀出锅。
- 营养亮点：蔬菜纤维足、饱腹感好；作为“蔬菜体积”补位更控热量。
- 注意事项：火候要大、时间要短；水分多会出汤影响口感；控油时用不粘锅或喷油更友好。
- 自动筛选依据：下厨房独立菜谱页；步骤极少、食材便宜易买；与 吃好/ 和 memory 去重索引无重复。

## 5. 青椒土豆丝

- 配图：
- 来源链接：https://www.xiachufang.com/recipe/18746/
- 推荐理由：土豆+青椒的经典快手下饭菜；切丝+快炒几分钟，材料便宜且稳定好买，适合日常晚餐。
- 适合场景：家常晚餐配菜 / 便当配菜 / 清淡快手素菜
- 预计耗时：10–20 分钟（取决于切丝速度）
- 主要食材：土豆、青椒/红椒（可选）、盐（可选：少量醋）
- 简明步骤：
  1) 土豆切丝后用清水冲洗/浸泡去淀粉，沥干；青椒切丝。
  2) 热锅少油，下土豆丝大火快炒至七分熟，加少许盐翻匀。
  3) 下青椒丝快速翻炒至断生即可（想更脆可在出锅前点几滴醋）。
- 营养亮点：饱腹感强、适合当“主食感配菜”；搭配一份蛋白质（鸡蛋/豆腐/肉类）更均衡。
- 注意事项：土豆丝泡洗可减少粘锅和发黑；青椒后下保持脆感；控油时避免长时间煸炒。
- 自动筛选依据：下厨房独立菜谱页；家常快手、步骤少、食材常见；与 吃好/ 和 memory 去重索引无重复。
