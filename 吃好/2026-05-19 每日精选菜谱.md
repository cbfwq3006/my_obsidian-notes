## 1. 蒜蓉油麦菜

- 配图：
- 推荐理由：典型“蒜香 + 大火快炒”的家常蔬菜，几分钟出锅；油麦菜口感脆嫩、体积大饱腹感强，少油也很香。
- 适合场景：工作日晚餐配菜 / 便当蔬菜位 / 两菜一汤中的快手青菜
- 预计耗时：8–12 分钟
- 主要食材：油麦菜、蒜、盐（可选：少许生抽/蚝油）
- 简明步骤：
  - 油麦菜洗净沥干，切段；蒜切末。
  - 热锅少油，蒜末小火爆香（别糊）。
  - 转大火下油麦菜快速翻炒至变软。
  - 加盐调味（可少许生抽/蚝油提鲜），翻匀出锅。
- 营养亮点：绿叶菜补充钾、叶酸与膳食纤维；少油快炒更适合日常控油。
- 注意事项：油麦菜入锅前尽量沥干，避免出水变“煮菜”；全程大火快炒更脆嫩。
- 来源链接：https://m.xiachufang.com/recipe/74776/

## 2. 肉末蒸豆腐

- 配图：
- 推荐理由：蒸菜路线几乎不费油，口感软嫩、下饭稳定；豆腐 + 肉末的组合蛋白质更完整，适合晚餐或便当。
- 适合场景：控油晚餐主菜 / 备餐（分装微波）/ 家庭餐桌“软烂好入口”
- 预计耗时：20–30 分钟
- 主要食材：嫩豆腐、猪肉末（或鸡肉末）、葱姜（可选）、生抽、盐
- 简明步骤：
  - 豆腐切块铺盘，略撒盐静置 5 分钟后倒掉渗水（可选）。
  - 肉末加生抽/盐（可加少量淀粉或蛋清更嫩，非必需）拌匀。
  - 肉末铺在豆腐上，水开上锅大火蒸 10–12 分钟。
  - 出锅淋少许生抽或热油（可选），撒葱花即可。
- 营养亮点：豆腐提供钙与植物蛋白，肉末补充铁与优质蛋白；蒸制更利于控油。
- 注意事项：嫩豆腐易碎，切块和摆盘轻拿轻放；蒸的时间别过长以免出水影响口感。
- 来源链接：https://m.xiachufang.com/recipe/101879959/

## 3. 家常豆腐（少油版思路）

- 配图：
- 推荐理由：用“煎/煎至定型 + 少量酱汁焖一会儿”的方式，少油也能有酱香；搭配青椒/胡萝卜/木耳等，营养更均衡。
- 适合场景：晚餐下饭主菜 / 便当主菜 / 想吃豆腐但不想太清淡
- 预计耗时：20–30 分钟
- 主要食材：老豆腐（北豆腐）、青椒（或彩椒）、胡萝卜（可选）、生抽、少量蚝油（可选）
- 简明步骤：
  - 老豆腐切片/块，厨房纸吸干表面水分。
  - 平底锅少油把豆腐两面煎至微黄，盛出。
  - 锅内留少许底油炒香蒜末/青椒，倒回豆腐。
  - 加生抽 + 少量热水（可加少许蚝油），小火焖 2–3 分钟入味即可。
- 营养亮点：豆腐高蛋白且含钙；配彩椒/胡萝卜可补充维C与胡萝卜素；整体油量可控。
- 注意事项：豆腐吸水后更不易溅油；酱汁别太咸，尽量“少量多次”调味。
- 来源链接：https://www.xiachufang.com/recipe/102221807/

## 4. 上汤娃娃菜（简化快手版）

- 配图：
- 推荐理由：娃娃菜本身甜口，做成“上汤”风味更鲜；用皮蛋/咸蛋/火腿等任一增鲜配件就能出味，适合忙碌日晚餐。
- 适合场景：两菜一汤的“汤位” / 清淡晚餐 / 配饭配面都合适
- 预计耗时：12–20 分钟
- 主要食材：娃娃菜、蒜（可选）、高汤/清水、（可选其一：皮蛋/咸蛋/火腿/虾皮）
- 简明步骤：
  - 娃娃菜对半切开再切段，洗净沥水。
  - 锅中少油爆香蒜末（可选），下配件略炒出香（如火腿/咸蛋）。
  - 加入清水或高汤煮开，下娃娃菜煮 3–5 分钟。
  - 加盐调味，出锅前可滴几滴香油（可选）。
- 营养亮点：娃娃菜低热量、补纤维；上汤做法汤水多、饱腹友好；配件可补蛋白与微量营养。
- 注意事项：娃娃菜别煮太久以免发黄变软；咸蛋/火腿自带咸度，盐要后放并减量。

## 5. 香菇青菜（5–10 分钟快炒）

- 配图：
- 推荐理由：香菇增鲜、青菜补纤维，一道就能把“蔬菜位”吃够；调味简单、锅气足，尤其适合作为便当蔬菜或晚餐搭配。
- 适合场景：工作日晚餐配菜 / 便当蔬菜位 / 轻食控油
- 预计耗时：8–15 分钟
- 主要食材：鲜香菇（或口蘑）、青菜（小油菜/上海青/青江菜等）、蒜、盐
- 简明步骤：
  - 香菇切片；青菜洗净沥干；蒜切末。
  - 热锅少油爆香蒜末，下香菇翻炒出香并略出汁。
  - 下青菜大火快炒至断生。
  - 加盐调味（可少许生抽提鲜），翻匀出锅。
- 营养亮点：菌菇提供多糖与矿物质、热量低；绿叶菜补充维生素与膳食纤维；少油快炒更轻负担。
- 注意事项：香菇先下锅更容易出香；青菜后下大火快炒保持脆嫩与颜色。
- 来源链接：https://m.xiachufang.com/recipe/104576147/
