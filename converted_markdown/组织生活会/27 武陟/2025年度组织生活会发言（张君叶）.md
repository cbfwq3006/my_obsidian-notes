2025年度组织生活会发言提纲

武陟党支部 张君叶

按照《关于召开基层党组织2025年度组织生活会和开展民主评议党员的通知》（焦联通党委〔2025〕号）要求，紧密结合工作实际，深入开展对照查摆工作，全面梳理工作中存在的问题，深挖问题产生的根源，从党性修养层面进行深刻剖析，并围绕问题制定了一系列具有针对性的改进措施。现对照检查发言如下：

一、上年度组织生活会整改落实情况

上年度组织生活会以来，我针对自身查摆出的问题，认真对照整改、逐项抓好落实，在同志们的帮助下，取得了一定进步。

一是理论学习有所加强。能够坚持参加集中学习、主动开展个人自学，对党的创新理论和公司业务政策做到常学常新，学习的主动性、系统性较以往有明显提升。

二是批评与自我批评更加认真。在日常工作和组织生活中，能够主动检视自身不足，对同事的问题也能坦诚提醒。

三是工作作风和执行能力持续改进。能够立足岗位履职尽责，对重点工作、重点项目做到盯紧抓实、闭环推进，工作态度更加端正，执行效率和落实质量稳步提升。

四是创新意识有所提升。在政企业务推进、客户服务、团队管理中，主动思考优化方法，积极探索更贴合实际的工作举措，创新开展工作的自觉性进一步增强。

深入学习贯彻中央八项规定精神学习教育开展情况

按照上级党组织和公司党委统一安排，我坚持学习中央八项规定精神，严格遵守公司各项纪律规定，结合联通政企工作实际，把作风建设融入日常业务全流程。

在客户对接、项目洽谈、业务合作中，始终坚持依规办事、廉洁从业，不搞违规接待、不接受不合理宴请、不触碰廉洁底线；在团队管理上，带头精简会议、简化流程、务实干事，把更多精力放在跑市场、抓项目、优服务、保指标上；坚决反对形式主义、官僚主义，不做表面文章、不搞花架子，切实以严实作风保障政企业务稳定发展。

落实巡视巡察和审计整改情况

2025年本人无巡视巡查和审计整改问题

四、问题查摆

（一）六个方面突出问题查摆

对照学习贯彻党的创新理论方面：学习存在碎片化、实用化倾向，对新思想、新论断、新要求系统研读不深、理解把握不透，与联通数字化转型、政企信息化、5G行业应用等工作结合不紧密，运用理论指导市场拓展、解决实际问题能力有待提升。

2. 加强党性锻炼方面：日常忙于业务工作，党性修养主动锤炼不够，自我要求有所放松，在面对困难任务、攻坚压力时，偶尔存在畏难情绪，斗争精神、担当精神还需进一步强化。

3. 联系服务群众方面：作为政企业务总监，更多聚焦大客户、重点项目、核心指标，对基层一线同事的困难诉求倾听不够，对中小客户服务延伸不够，主动靠前、上门服务的意识和行动仍有不足。

4. 发挥先锋模范作用方面：能够完成本职工作，但标杆意识、带头作用发挥不充分，在急难险重任务面前，带头冲、带头干的劲头不足，带领团队攻坚克难、争先创优的凝聚力、战斗力有待加强。

5. 改作风树新风方面：工作中仍存在重部署、轻落实，重结果、轻过程的情况，对一些细节把控不严，跟踪督办不够到位；勤俭节约、过紧日子的思想树得不够牢，在办公耗材、业务成本管控上还可更精细。

6. 宗教信仰以及参加迷信活动方面：本人坚定理想信念，无任何宗教信仰，从不参与任何迷信活动，始终坚持唯物主义立场，严守党员政治纪律和政治规矩。

（二）需要说明和报告的事项

1.个人需报告的重大事项和有关问题。

（1）配偶在武陟本地工作，子女是在校学生，配偶子女无经商办企业情况。

（2）个人房产情况、办公用房是否超标。

个人房产一处，办公室和政企中心人员集中办公，未超标。

2.群众反映、巡视反馈、组织约谈函询及受到问责等情况说明。

2025年无群众反映、巡视反馈、组织约谈函询等问题，未受到问责。

五、原因剖析

1、思想认识站位不高：对理论学习、作风建设、党性锤炼的重要性认识不足，没有真正把学习和改造主观世界、推动政企工作结合起来，思想上存在重业务、轻学习的倾向。

2、责任担当意识弱化：满足于完成基本任务、守住不出问题底线，主动谋事、大胆干事、争先创优的进取心有所减退，面对复杂市场环境和激烈竞争，缺乏迎难而上的闯劲。

3、工作作风不够严实：存在一定惰性思维，统筹协调、闭环执行不够到位，深入一线、深入客户、深入现场不够，导致部分工作推进不细、落实不实。

4. 自我要求标准不严：日常自我监督、自我约束有所放松，批评与自我批评武器用得不够经常，对自身问题反思不深、整改不彻底，长效提升机制没有真正建立。

六、整改措施和努力方向

1、强化理论武装，提升学用结合能力：坚持把政治学习放在首位，固定每天抽出1小时学习时间、细化学习内容，把党的创新理论与联通政企改革、市场攻坚、项目运营、团队管理紧密结合，做到学深悟透、学以致用，切实以理论指导业务实践。

2、严肃党内生活，用好批评武器：主动参加组织生活，敢于刀刃向内、深刻自我批评，不断提升自我净化、自我完善能力。

3、锤炼严实作风，提高执行落实质效：立足工作职责，聚焦政企指标、重点项目、客户服务，清单化管理、项目化推进、闭环式落实，带头深入一线、跑办对接、攻坚克难，以钉钉子精神把各项工作抓细抓实抓出成效。

4、勇于担当创新，增强争先创优意识：主动适应联通数字化转型和政企市场新变化，打破传统思维、创新工作方法，在行业解决方案、新兴业务拓展、客户价值提升上大胆探索，带领团队争先进、创亮点、多创收。

5、严守纪律规矩，筑牢廉洁从业底线：持续深入落实中央八项规定精神，严格遵守公司廉洁从业各项规定，清白做人、干净做事，管好自己、带好队伍，以优良作风树立联通政企战线党员良好形象。

今后，我将以此次组织生活会为新起点，正视问题、立行立改，担当作为、真抓实干，为公司高质量发展贡献更大力量。

发言完毕，谢谢大家。
