# 拿走不谢！用Obsidian搭一套从工作到生活的完整的任务管理库

**原文链接**：[[笔记同步助手/2026-05-11/拿走不谢！用Obsidian搭一套从工作到生活的完整的任务管理库]]

## 核心观点

用Obsidian搭建PTV（Personal Task Vault）个人任务宝库，**核心思路：把日-周-月-年串起来，让任务自动跟着时间流动**。解决笔记和任务两个系统互不打通的根本问题。

## 系统架构

### 日-周-月-年四级联动

| 层级 | 内容 | 跳转关系 |
|------|------|---------|
| 日记 | 今天做了什么 | → 本周周记、本月月记、今年年记 |
| 周记 | 这周完成情况 | → 本周七天日记、本月月记 |
| 月记 | 本月目标进度 | → 各周周记、今年年记 |
| 年记 | 全年目标 | → 各月月记 |

**关键机制**：Tasks插件自动汇总——周记和月记里的查询代码，会自动把所有笔记里标了本周到期的任务抓取过来显示。

### 每日三段式使用法

**早晨（3分钟）**：写"今日重点"（1-3件必须做完的事），列今日任务（工作+个人分开）

**白天（随时）**：突然想起的事直接扔进"随手记"或`07-收集箱`，不打断当前工作

**睡前（5分钟）**：
1. 勾掉已完成任务
2. 未完成的加明天截止日期（自动出现在明天日记）
3. 写一行"明日预览"

### 周记（周末15分钟）

1. 扫一眼自动汇总的未完成任务
2. 写"完成了什么"（两三行）
3. 没完成的：延期或删掉
4. 写一句本周感悟

**核心原则**：敢于删掉不重要的任务比硬撑着做完所有事重要得多。拖了三周没动的事，大概率不重要，删了就删了。

### 月记（月末30分钟）

1. 看当月目标完成了多少
2. 更新各周状态
3. **手动迁移**：把没做完但确实需要做的事，复制到下个月

**刻意保留"抄一遍"的动作**：抄的时候会认真想"这个事我是不是非做不可"，自动过滤不重要的事。

### 年记（年初+年底）

年初写5个维度目标：

| 维度 | 问自己 |
|------|------|
| 工作 | 今年想做成哪件事？ |
| 学习 | 想学什么新东西？ |
| 健康 | 身体和习惯的目标？ |
| 财务 | 收入和储蓄有什么计划？ |
| 关系 | 想维护哪些人？ |

### 收集箱（焦虑的出口）

任何念头冒出来→3秒扔进收集箱→脑子轻松→周末10分钟集中处理

## 系统优势

| 优点 | 实际感受 |
|------|---------|
| 打通笔记和任务 | 读文章时写的"下周试试"，任务就在笔记旁边 |
| 级联自动追踪 | 今天写的待办，周末自动出现在周记里 |
| 四个维度全覆盖 | 从每天琐事到每年目标，一个库全管 |
| 纯本地不过期 | 数据是.md文件，二十年后打开还能用 |
| 免费无订阅 | Obsidian免费，插件免费 |

## 每日执行方法（用户要求按此执行）

```
早晨：打开今日日记 → 写今日重点(1-3件) → 列今日任务
白天：随手记 / 收集箱
睡前：勾完成 → 未完成加明日截止 → 写明日预览
周末：15分钟周复盘
月末：30分钟月复盘+手动迁移
```
